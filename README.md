# 🚀 TrendPulse Project

## 📌 Overview

TrendPulse is a data pipeline project that fetches, processes, analyzes, and visualizes trending stories from HackerNews.

---

## 🧩 Project Structure

### ✅ Task 1: Data Collection

* Fetches top stories from HackerNews API
* Categorizes into:

  * Technology
  * World News
  * Sports
  * Science
  * Entertainment
* Saves data as JSON

### ✅ Task 2: Data Cleaning

* Cleans and processes raw JSON data
* Converts to structured CSV format

### ✅ Task 3: Data Analysis

* Uses Pandas & NumPy
* Performs statistical analysis

### ✅ Task 4: Data Visualization

* Generates visual insights:

  * Score distribution
  * Category comparison
  * Top stories

---

## 📁 Output Files

* JSON data
* Cleaned CSV
* Analysis report
* Visualizations (PNG)

---

## 🛠️ Technologies Used

* Python
* Requests
* Pandas
* NumPy
* Matplotlib

---

## ▶️ How to Run

```bash
python task1_data_collection.py
python task2_data_cleaning.py
python task3_analysis.py
python task4_visualization.py
```
